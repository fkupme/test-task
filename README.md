# MUSbooking Admin Panel

Административная панель для управления сервисом MUSbooking.

## Текущее задание

### Раздел: Типы позиций

#### Экран: Таблица с типами позиций




**Основные элементы:**

1. **Поиск** - строка поиска по типам позиций

2. **Фильтр по сферам** - множественный выбор сфер для фильтрации
   - Пример тегов в UI Kit

3. **Таблица из 2 вкладок:**
   - Оборудование
   - Услуги
   
   Обе таблицы содержат следующие колонки:
   - Тип оборудования или тип услуги
   - Сфера (множественное кол-во тегов) - пример тегов в UI Kit
   - Признак видимости (иконкой)

#### Требования к оформлению

**Цветовая схема:** использовать нейтральные цвета

**Визуальное отделение:**
- Визуальное отделение видимости типа оборудования или услуги (цвет текста "Secondary")

#### Пользовательские сценарии

**Сценарий 1: Загрузка страницы и просмотр**

1. Пользователь открывает страницу с таблицей, подгружаются скелетоны, затем сама таблица

**Сценарий 2: Добавление типа оборудования**

1. Пользователь нажимает кнопку "добавить"
2. Появляется панель управления перекрывая таблицу
3. Панель управления содержит следующие поля:
   - Заголовок "Добавление типа оборудования" с возможностью закрыть
   - Поле "Название"
   - Поле с выбором сфер, к которым будет применен тип оборудования (множественный выбор)
   - Статус (с выбором параметра: отображать/скрыть)
   - Неактивную кнопку "Сохранить"
4. Пользователь заполняет поля
5. Выбирает параметр видимости
6. Кнопка сохранить становится активной
7. Пользователь сохраняет изменение, тип оборудования/услуги появляется в таблице

#### Дополнительные требования

1. Адаптив под заданные экраны (Desktop, Laptop, Tablet)

## Компоненты UI

### Таблица

Компонент таблицы находится в:
- [components/UI/table/DataTable.vue](components/UI/table/DataTable.vue)
- [Документация компонента таблицы](components/UI/table/README.md)

### Другие UI компоненты

Все UI компоненты находятся в директории `components/UI/`:
- `cards/` - карточки
- `controls/` - кнопки, селекты, переключатели
- `display/` - компоненты отображения
- `fields/` - поля ввода
- `filters/` - фильтры
- `icons/` - иконки
- `layout/` - компоненты разметки
- `notifications/` - уведомления
- `pickers/` - пикеры (выбор даты, времени и т.д.)
- `table/` - таблицы

## API

### PartnerZonesService

Документация по API методам для работы с зонами партнеров:
- [PartnerZonesService API Documentation](docs/PartnerZonesService.md)

**Доступные методы:**
- `getPositionGroupsWeb()` - получение групп позиций
- `createPositionGroup()` - создание группы позиций
- `updatePositionGroup()` - обновление группы позиций
- `getSpheres()` - получение списка сфер

## Структура проекта

- `classes/api/` - API сервисы
- `components/` - Vue компоненты
- `composables/` - переиспользуемая логика
- `constants/` - константы
- `layouts/` - макеты страниц
- `middleware/` - middleware для роутинга
- `pages/` - страницы приложения
- `plugins/` - плагины Nuxt
- `stores/` - Pinia stores
- `types/` - TypeScript типы
- `utils/` - утилиты
